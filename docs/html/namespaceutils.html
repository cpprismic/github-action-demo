◆ formatUptime()

std::string utils::formatUptime ( double hours)

Форматировать время работы системы в читаемый вид.

Пример: 25.5 часа → "1d 1h 30m"

Аргументы
hoursВремя работы в часах (>= 0).
Возвращает
Строка вида "Xd Yh Zm" (секция дней опускается при 0).

◆ parseArguments()

bool utils::parseArguments ( int argc,
char * argv[],
bool & continuous_mode,
int & interval )

Разобрать аргументы командной строки.

Поддерживаемые опции:

  • (без аргументов) — однократный запуск;
  • -c / --continuous [N] — непрерывный режим с интервалом N секунд;
  • -h / --help — вывод справки.
Аргументы
argcКоличество аргументов.
argvМассив аргументов.
[out]continuous_modetrue, если запрошен непрерывный режим.
[out]intervalИнтервал в секундах (по умолчанию 2).
Возвращает
true при успешном разборе, false при ошибке (сообщение выведено в stderr).
