Подробное описание

Монитор системных ресурсов Linux.

Читает данные из файловой системы /proc и предоставляет информацию о загрузке процессора, использовании памяти и времени работы системы. Поддерживает запуск внутри Docker-контейнера с примонтированным /host/proc.

Копирование запрещено — объект владеет внутренним состоянием CPU-дифференциала.
